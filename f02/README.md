# Lógica de Programação e Git/GitHub

### Fundamentos de Programação, Python e Versionamento

Nesta fase foram estudados os fundamentos da lógica de programação utilizando Python, além das principais ferramentas de versionamento de código com Git e GitHub.
Também foram desenvolvidas habilidades para criação de programas, manipulação de arquivos, automação básica e gerenciamento de projetos utilizando controle de versão.

## Conteúdos Abordados:

#### Lógica de Programação
* Algoritmos
* Pseudocódigo
* Resolução de problemas
* Fluxo lógico

#### Variáveis e Tipos de Dados
* Variáveis
* Tipos primitivos
* Operações matemáticas
* Conversão de tipos

#### Estruturas de Decisão
* `if`
* `elif`
* `else`
* Operadores de comparação
* Operadores lógicos (`and`, `or`, `not`)
