# Cybersecurity Training

Repositório dedicado para documentar minha jornada de aprendizado em Cybersecurity.

Aqui estão registrados os laboratórios, desafios, anotações e conceitos estudados durante minha formação, com foco em fundamentos de computação, Linux, redes, segurança da informação e práticas de análise de segurança.

## Objetivos

* Desenvolver conhecimentos em Cybersecurity.
* Praticar conceitos por meio de laboratórios, provas e desafios.
* Registrar aprendizados e evolução técnica.

Cada fase contém resumos dos conteúdos estudados, comandos utilizados e conceitos aprendidos.

⚠️ Projeto desenvolvido para estudo e prática utilizando uma plataforma especifica de treinamentos.
